## Installation

```bash
  pip install -r requirements.txt
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file.

`MYSQL_HOST`

`MYSQL_PORT`

`MYSQL_DATABASE`

`MYSQL_USERNAME`

`MYSQL_PASSWORD`

`INTERVALO_BACKUP_LOGGER`

`ENABLE_LOGS`

`FLASK_PORT`

`FLASK_DEBUG`

`FLASK_ENDPOINT`

`SECRET_KEY`

`JWT_SECRET_KEY`

`JWT_ALGORITHM`

`JWT_TIME`
